Javascript жЁЎеқ—еҢ–зј–зЁӢ-require.js зҡ„з”Ёжі•

еҸ‘иЎЁдәҺ 2017-08-28 | жӣҙж–°дәҺ 2018-12-26 | еҲҶзұ»дәҺ JavaScript

JavascriptжЁЎеқ—еҢ–зј–зЁӢпјҢе·Із»ҸжҲҗдёәдёҖдёӘиҝ«еҲҮзҡ„йңҖжұӮгҖӮзҗҶжғіжғ…еҶөдёӢпјҢејҖеҸ‘иҖ…еҸӘйңҖиҰҒе®һзҺ°ж ёеҝғзҡ„дёҡеҠЎйҖ»иҫ‘пјҢе…¶д»–йғҪеҸҜд»ҘеҠ иҪҪеҲ«дәәе·Із»ҸеҶҷеҘҪзҡ„жЁЎеқ—гҖӮ
дҪҶжҳҜпјҢJavascriptдёҚжҳҜдёҖз§ҚжЁЎеқ—еҢ–зј–зЁӢиҜӯиЁҖпјҢе®ғдёҚж”ҜжҢҒзұ»(class)пјҢжӣҙеҲ«жҸҗжЁЎеқ—дәҶпјҢжң¬ж–Үе°ҶдҪҝз”Ёrequire.jsе®һзҺ°вҖңJavaScriptжЁЎеқ—еҢ–зј–зЁӢвҖқгҖӮ

require.jsжҳҜдёҖдёӘйқһеёёжөҒиЎҢзҡ„еә“гҖӮ

дёҖгҖҒдёәд»Җд№ҲиҰҒз”Ёrequire.jsпјҹ

жҲ‘д»¬дёҖиҲ¬йғҪд№ жғҜе°ҶJavascriptд»Јз ҒйғҪеҶҷеңЁдёҖдёӘж–Үд»¶йҮҢйқўпјҢеҸӘиҰҒеҠ иҪҪиҝҷдёҖдёӘж–Үд»¶е°ұеӨҹдәҶгҖӮеҗҺжқҘпјҢд»Јз Ғи¶ҠжқҘи¶ҠеӨҡпјҢйҖ»иҫ‘и¶ҠжқҘи¶ҠеӨҚжқӮпјҢдёҖдёӘж–Үд»¶дёҚеӨҹдәҶпјҢеҝ…йЎ»еҲҶжҲҗеӨҡдёӘж–Үд»¶пјҢдҫқж¬ЎеҠ иҪҪгҖӮйӮЈе°ұеҮәзҺ°дәҶдёӢйқўзҡ„д»Јз Ғпјҡ

иҝҷж®өд»Јз Ғдҫқж¬ЎеҠ иҪҪеӨҡдёӘjsж–Үд»¶гҖӮ

иҝҷж ·зҡ„еҶҷжі•жңүеҫҲеӨ§зҡ„зјәзӮ№гҖӮйҰ–е…ҲпјҢеҠ иҪҪзҡ„ж—¶еҖҷпјҢжөҸи§ҲеҷЁдјҡеҒңжӯўзҪ‘йЎөжёІжҹ“пјҢеҠ иҪҪж–Үд»¶и¶ҠеӨҡпјҢзҪ‘йЎөеӨұеҺ»е“Қеә”зҡ„ж—¶й—ҙе°ұдјҡи¶Ҡй•ҝпјӣе…¶ж¬ЎпјҢз”ұдәҺjsж–Үд»¶д№Ӣй—ҙеӯҳеңЁдҫқиө–е…ізі»пјҢеӣ жӯӨеҝ…йЎ»дёҘж јдҝқиҜҒеҠ иҪҪйЎәеәҸпјҲжҜ”еҰӮдёҠдҫӢзҡ„1.jsиҰҒеңЁ2.jsзҡ„еүҚйқўпјүпјҢдҫқиө–жҖ§жңҖеӨ§зҡ„жЁЎеқ—дёҖе®ҡиҰҒж”ҫеҲ°жңҖеҗҺеҠ иҪҪпјҢеҪ“дҫқиө–е…ізі»еҫҲеӨҚжқӮзҡ„ж—¶еҖҷпјҢд»Јз Ғзҡ„зј–еҶҷе’Ңз»ҙжҠӨйғҪдјҡеҸҳеҫ—еӣ°йҡҫгҖӮ

require.jsзҡ„иҜһз”ҹпјҢе°ұжҳҜдёәдәҶи§ЈеҶіиҝҷдёӨдёӘй—®йўҳпјҡ

пјҲ1пјүе®һзҺ°jsж–Үд»¶зҡ„ејӮжӯҘеҠ иҪҪпјҢйҒҝе…ҚзҪ‘йЎөеӨұеҺ»е“Қеә”пјӣ

пјҲ2пјүз®ЎзҗҶжЁЎеқ—д№Ӣй—ҙзҡ„дҫқиө–жҖ§пјҢдҫҝдәҺд»Јз Ғзҡ„зј–еҶҷе’Ңз»ҙжҠӨгҖӮ

дәҢгҖҒrequire.jsзҡ„еҠ иҪҪ

дҪҝз”Ёrequire.jsзҡ„з¬¬дёҖжӯҘпјҢжҳҜе…ҲеҺ»е®ҳж–№зҪ‘з«ҷдёӢиҪҪжңҖж–°зүҲжң¬гҖӮ

дёӢиҪҪеҗҺпјҢеҒҮе®ҡжҠҠе®ғж”ҫеңЁjsеӯҗзӣ®еҪ•дёӢйқўпјҢе°ұеҸҜд»ҘеҠ иҪҪдәҶгҖӮ

еҠ иҪҪиҝҷдёӘж–Үд»¶пјҢеҸҜиғҪйҖ жҲҗзҪ‘йЎөеӨұеҺ»е“Қеә”гҖӮи§ЈеҶіеҠһжі•жңүдёӨдёӘпјҢдёҖдёӘжҳҜжҠҠе®ғж”ҫеңЁзҪ‘йЎөеә•йғЁеҠ иҪҪпјҢеҸҰдёҖдёӘжҳҜеҶҷжҲҗдёӢйқўиҝҷж ·пјҡ

asyncеұһжҖ§иЎЁжҳҺиҝҷдёӘж–Үд»¶йңҖиҰҒејӮжӯҘеҠ иҪҪпјҢйҒҝе…ҚзҪ‘йЎөеӨұеҺ»е“Қеә”гҖӮIEдёҚж”ҜжҢҒиҝҷдёӘеұһжҖ§пјҢеҸӘж”ҜжҢҒdeferпјҢжүҖд»ҘжҠҠdeferд№ҹеҶҷдёҠгҖӮ

еҠ иҪҪrequire.jsд»ҘеҗҺпјҢдёӢдёҖжӯҘе°ұиҰҒеҠ иҪҪжҲ‘д»¬иҮӘе·ұзҡ„д»Јз ҒдәҶгҖӮеҒҮе®ҡжҲ‘д»¬иҮӘе·ұзҡ„д»Јз Ғж–Үд»¶жҳҜmain.jsпјҢд№ҹж”ҫеңЁjsзӣ®еҪ•дёӢйқўгҖӮйӮЈд№ҲпјҢеҸӘйңҖиҰҒеҶҷжҲҗдёӢйқўиҝҷж ·е°ұиЎҢдәҶпјҡ

data-mainеұһжҖ§зҡ„дҪңз”ЁжҳҜпјҢжҢҮе®ҡзҪ‘йЎөзЁӢеәҸзҡ„дё»жЁЎеқ—гҖӮеңЁдёҠдҫӢдёӯпјҢе°ұжҳҜjsзӣ®еҪ•дёӢйқўзҡ„main.jsпјҢиҝҷдёӘж–Үд»¶дјҡз¬¬дёҖдёӘиў«require.jsеҠ иҪҪгҖӮз”ұдәҺrequire.jsй»ҳи®Өзҡ„ж–Үд»¶еҗҺзјҖеҗҚжҳҜjsпјҢжүҖд»ҘеҸҜд»ҘжҠҠmain.jsз®ҖеҶҷжҲҗmainгҖӮ

дёүгҖҒдё»жЁЎеқ—зҡ„еҶҷжі•

дёҠйқўзҡ„main.jsпјҢжҲ‘жҠҠе®ғз§°дёәвҖқдё»жЁЎеқ—вҖқпјҢж„ҸжҖқжҳҜж•ҙдёӘзҪ‘йЎөзҡ„е…ҘеҸЈд»Јз ҒгҖӮе®ғжңүзӮ№еғҸCиҜӯиЁҖзҡ„main()еҮҪж•°пјҢжүҖжңүд»Јз ҒйғҪд»Һиҝҷе„ҝејҖе§ӢиҝҗиЎҢгҖӮ

дёӢйқўе°ұжқҘзңӢпјҢжҖҺд№ҲеҶҷmain.jsгҖӮ

еҰӮжһңжҲ‘д»¬зҡ„д»Јз ҒдёҚдҫқиө–д»»дҪ•е…¶д»–жЁЎеқ—пјҢйӮЈд№ҲеҸҜд»ҘзӣҙжҺҘеҶҷе…Ҙjavascriptд»Јз ҒгҖӮ

// main.js

alert(вҖңеҠ иҪҪжҲҗеҠҹпјҒвҖқ);

дҪҶиҝҷж ·зҡ„иҜқпјҢе°ұжІЎеҝ…иҰҒдҪҝз”Ёrequire.jsдәҶгҖӮзңҹжӯЈеёёи§Ғзҡ„жғ…еҶөжҳҜпјҢдё»жЁЎеқ—дҫқиө–дәҺе…¶д»–жЁЎеқ—пјҢиҝҷж—¶е°ұиҰҒдҪҝз”ЁAMDи§„иҢғе®ҡд№үзҡ„зҡ„require()еҮҪж•°гҖӮ

require()еҮҪж•°жҺҘеҸ—дёӨдёӘеҸӮж•°гҖӮз¬¬дёҖдёӘеҸӮж•°жҳҜдёҖдёӘж•°з»„пјҢиЎЁзӨәжүҖдҫқиө–зҡ„жЁЎеқ—пјҢдёҠдҫӢе°ұжҳҜ[вҖҳmoduleAвҖҷ, вҖҳmoduleBвҖҷ, вҖҳmoduleCвҖҷ]пјҢеҚідё»жЁЎеқ—дҫқиө–иҝҷдёүдёӘжЁЎеқ—пјӣз¬¬дәҢдёӘеҸӮж•°жҳҜдёҖдёӘеӣһи°ғеҮҪж•°пјҢеҪ“еүҚйқўжҢҮе®ҡзҡ„жЁЎеқ—йғҪеҠ иҪҪжҲҗеҠҹеҗҺпјҢе®ғе°Ҷиў«и°ғз”ЁгҖӮеҠ иҪҪзҡ„жЁЎеқ—дјҡд»ҘеҸӮж•°еҪўејҸдј е…ҘиҜҘеҮҪж•°пјҢд»ҺиҖҢеңЁеӣһи°ғеҮҪж•°еҶ…йғЁе°ұеҸҜд»ҘдҪҝз”ЁиҝҷдәӣжЁЎеқ—гҖӮ

require()ејӮжӯҘеҠ иҪҪmoduleAпјҢmoduleBе’ҢmoduleCпјҢжөҸи§ҲеҷЁдёҚдјҡеӨұеҺ»е“Қеә”пјӣе®ғжҢҮе®ҡзҡ„еӣһи°ғеҮҪж•°пјҢеҸӘжңүеүҚйқўзҡ„жЁЎеқ—йғҪеҠ иҪҪжҲҗеҠҹеҗҺпјҢжүҚдјҡиҝҗиЎҢпјҢи§ЈеҶідәҶдҫқиө–жҖ§зҡ„й—®йўҳгҖӮ

дёӢйқўпјҢжҲ‘д»¬зңӢдёҖдёӘе®һйҷ…зҡ„дҫӢеӯҗгҖӮ

еҒҮе®ҡдё»жЁЎеқ—дҫқиө–jqueryгҖҒunderscoreе’ҢbackboneиҝҷдёүдёӘжЁЎеқ—пјҢmain.jsе°ұеҸҜд»Ҙиҝҷж ·еҶҷпјҡ

require.jsдјҡе…ҲеҠ иҪҪjQueryгҖҒunderscoreе’ҢbackboneпјҢз„¶еҗҺеҶҚиҝҗиЎҢеӣһи°ғеҮҪж•°гҖӮдё»жЁЎеқ—зҡ„д»Јз Ғе°ұеҶҷеңЁеӣһи°ғеҮҪж•°дёӯгҖӮ

еӣӣгҖҒжЁЎеқ—зҡ„еҠ иҪҪ

дёҠдёҖиҠӮжңҖеҗҺзҡ„зӨәдҫӢдёӯпјҢдё»жЁЎеқ—зҡ„дҫқиө–жЁЎеқ—жҳҜ[вҖҳjqueryвҖҷ, вҖҳunderscoreвҖҷ, вҖҳbackboneвҖҷ]гҖӮй»ҳи®Өжғ…еҶөдёӢпјҢrequire.jsеҒҮе®ҡиҝҷдёүдёӘжЁЎеқ—дёҺmain.jsеңЁеҗҢдёҖдёӘзӣ®еҪ•пјҢж–Үд»¶еҗҚеҲҶеҲ«дёәjquery.jsпјҢunderscore.jsе’Ңbackbone.jsпјҢз„¶еҗҺиҮӘеҠЁеҠ иҪҪгҖӮ

дҪҝз”Ёrequire.config()ж–№жі•пјҢжҲ‘д»¬еҸҜд»ҘеҜ№жЁЎеқ—зҡ„еҠ иҪҪиЎҢдёәиҝӣиЎҢиҮӘе®ҡд№үгҖӮrequire.config()е°ұеҶҷеңЁдё»жЁЎеқ—пјҲmain.jsпјүзҡ„еӨҙйғЁгҖӮеҸӮж•°е°ұжҳҜдёҖдёӘеҜ№иұЎпјҢиҝҷдёӘеҜ№иұЎзҡ„pathsеұһжҖ§жҢҮе®ҡеҗ„дёӘжЁЎеқ—зҡ„еҠ иҪҪи·Ҝеҫ„гҖӮ

дёҠйқўзҡ„д»Јз Ғз»ҷеҮәдәҶдёүдёӘжЁЎеқ—зҡ„ж–Үд»¶еҗҚпјҢи·Ҝеҫ„й»ҳи®ӨдёҺmain.jsеңЁеҗҢдёҖдёӘзӣ®еҪ•пјҲjsеӯҗзӣ®еҪ•пјүгҖӮеҰӮжһңиҝҷдәӣжЁЎеқ—еңЁе…¶д»–зӣ®еҪ•пјҢжҜ”еҰӮjs/libзӣ®еҪ•пјҢеҲҷжңүдёӨз§ҚеҶҷжі•гҖӮдёҖз§ҚжҳҜйҖҗдёҖжҢҮе®ҡи·Ҝеҫ„гҖӮ

еҸҰдёҖз§ҚеҲҷжҳҜзӣҙжҺҘж”№еҸҳеҹәзӣ®еҪ•пјҲbaseUrlпјүгҖӮ

еҰӮжһңжҹҗдёӘжЁЎеқ—еңЁеҸҰдёҖеҸ°дё»жңәдёҠпјҢд№ҹеҸҜд»ҘзӣҙжҺҘжҢҮе®ҡе®ғзҡ„зҪ‘еқҖпјҢжҜ”еҰӮпјҡ

дә”гҖҒAMDжЁЎеқ—зҡ„еҶҷжі•

require.jsеҠ иҪҪзҡ„жЁЎеқ—пјҢйҮҮз”ЁAMDи§„иҢғгҖӮд№ҹе°ұжҳҜиҜҙпјҢжЁЎеқ—еҝ…йЎ»жҢүз…§AMDзҡ„и§„е®ҡжқҘеҶҷгҖӮ

е…·дҪ“жқҘиҜҙпјҢе°ұжҳҜжЁЎеқ—еҝ…йЎ»йҮҮз”Ёзү№е®ҡзҡ„define()еҮҪж•°жқҘе®ҡд№үгҖӮеҰӮжһңдёҖдёӘжЁЎеқ—дёҚдҫқиө–е…¶д»–жЁЎеқ—пјҢйӮЈд№ҲеҸҜд»ҘзӣҙжҺҘе®ҡд№үеңЁdefine()еҮҪж•°д№ӢдёӯгҖӮ

еҒҮе®ҡзҺ°еңЁжңүдёҖдёӘmath.jsж–Үд»¶пјҢе®ғе®ҡд№үдәҶдёҖдёӘmathжЁЎеқ—гҖӮйӮЈд№ҲпјҢmath.jsе°ұиҰҒиҝҷж ·еҶҷпјҡ

еҠ иҪҪж–№жі•еҰӮдёӢпјҡ

еҰӮжһңиҝҷдёӘжЁЎеқ—иҝҳдҫқиө–е…¶д»–жЁЎеқ—пјҢйӮЈд№Ҳdefine()еҮҪж•°зҡ„з¬¬дёҖдёӘеҸӮж•°пјҢеҝ…йЎ»жҳҜдёҖдёӘж•°з»„пјҢжҢҮжҳҺиҜҘжЁЎеқ—зҡ„дҫқиө–жҖ§гҖӮ

еҪ“require()еҮҪж•°еҠ иҪҪдёҠйқўиҝҷдёӘжЁЎеқ—зҡ„ж—¶еҖҷпјҢе°ұдјҡе…ҲеҠ иҪҪmyLib.jsж–Үд»¶гҖӮ

е…ӯгҖҒеҠ иҪҪйқһи§„иҢғзҡ„жЁЎеқ—

зҗҶи®әдёҠпјҢrequire.jsеҠ иҪҪзҡ„жЁЎеқ—пјҢеҝ…йЎ»жҳҜжҢүз…§AMDи§„иҢғгҖҒз”Ёdefine()еҮҪж•°е®ҡд№үзҡ„жЁЎеқ—гҖӮдҪҶжҳҜе®һйҷ…дёҠпјҢиҷҪз„¶е·Із»ҸжңүдёҖйғЁеҲҶжөҒиЎҢзҡ„еҮҪж•°еә“пјҲжҜ”еҰӮjQueryпјүз¬ҰеҗҲAMDи§„иҢғпјҢжӣҙеӨҡзҡ„еә“е№¶дёҚз¬ҰеҗҲгҖӮйӮЈд№ҲпјҢrequire.jsжҳҜеҗҰиғҪеӨҹеҠ иҪҪйқһи§„иҢғзҡ„жЁЎеқ—е‘ўпјҹ

еӣһзӯ”жҳҜеҸҜд»Ҙзҡ„гҖӮ
